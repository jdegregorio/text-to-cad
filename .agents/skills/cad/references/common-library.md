# Shared CAD Library

`scripts/common` contains shared library code for CAD source discovery, generator envelope metadata parsing, STEP render artifacts, assembly flattening, STEP scene/selector extraction, STL/GLB/DXF helpers, and validation helpers.

It is not a generator command. Use `scripts/gen_step_part/cli.py`, `scripts/gen_step_assembly/cli.py`, or `scripts/gen_dxf/cli.py` for generation.
